Variables and Strings in Java

What are Variables?

  • A variable is a container that holds data.
  • In Java, variables must have a specific data type (e.g., int, double, String).
  • Variables are used to store values for later use in a program.

What is a String?

  • A String is a sequence of characters enclosed in double quotes.
  • Strings in Java are objects of the String class.
  • They are immutable, meaning their values cannot be changed once created.

How to Declare Variables and Strings

// Declaring an integer variable

int age = 25;

// Declaring a string variable

String name = "Roha";
